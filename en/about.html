About

GFork Proxy is a Greasy Fork mirror acceleration service, helping users access and download user scripts faster and more reliably.

Key Features

  • Multi-CDN acceleration — scripts and resources served through optimized Asia-Pacific edge nodes
  • Advanced search — filter by installs, ratings, date, language, domain
  • Download accelerator — automatically selects the fastest CDN node
  • Multi-language support — 简体中文, 繁體中文, English, 日本語

Open Source

This project is open source. All code is available on GitHub under the GPLv3 license.
